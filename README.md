# Atnest

Basic Angular project.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or v20 recommended)
- npm (comes with Node.js)

## Setup

```bash
cd atnest
npm install
```

## Run

```bash
npm start
```

Then open http://localhost:4200 in your browser.

## Build

```bash
npm run build
```

Output is in `dist/atnest`.

## Test

```bash
npm test
```
